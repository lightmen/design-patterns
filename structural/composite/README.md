# 组合模式(Composite)



## 定义

将对象组合成树形结构以表示“部分-整体”的层次结构。Composite是的用户对单个对象和组合对象的使用具有一致性



## 参与者

- Component
  - 为组合对象的对象声明接口
  - 在适当的情况下，实现所有类共有接口的缺省行为
  - 声明一个接口用于访问和管理Component的子组件
  - （可选）在递归接口中定义一个接口，用于访问一个父部件，并在合适的情况下实现它
- Leaf
  - 在组合中表示叶节点对象，叶节点没有子节点
  - 在组合中定义图元对象的行为
- Composite
  - 定义有子部件的那些部件的行为
  - 存储子部件
  - 在Component接口中实现操纵组合部件的对象
